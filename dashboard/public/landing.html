Protocol

Features

What powers the protocol

💸

USDC Pay-Per-Request

Every API call is settled instantly in USDC on Base via x402 payment rails. Automated settlement — no billing infrastructure needed.

🔒

Self-Hosted & Secure

Your API keys never leave your server. Endpoint ownership verified via challenge-response without centralized intermediaries.

🔍

Agent-Native Discovery

AI agents discover your API naturally through the marketplace, MCP server integration, and skill.md — no human in the loop required.

⚡

Instant Onboarding

Deploy the open-source Router with CDP Facilitator support and start accepting L2 payments on Base in minutes, not days.

🛡️

Built-In Encryption

SKALE/BITE encryption protects your agent's intent and conditions until finality is reached on-chain. MEV-resistant by design.

📋

AP2 Mandate Safety

Enforces hard spend limits, expiry, and strict tool allowlists to protect from rogue agents and runaway costs — signed by the user.

🔗

Chained API Calls

Discovery of marketplace endpoints to fulfill complex tasks with multiple steps and chained calls from separate providers in sequence.

🔁

x402 Upstream Checker

Built-in protection against unauthorized markup. Verifies upstream errors to trigger refund rules — agents only pay for successful calls.

🧾

Transaction Receipts

Every request generates a JSON receipt with request hash binding and idempotency protection, linked to Base for your audit trail.
